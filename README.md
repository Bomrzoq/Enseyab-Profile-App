### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/flutter/flutter.git -b stable
   ```
3. Edit [environment variables](https://stackoverflow.com/questions/60212125/flutter-install-in-windows-environment-variable) for your account 
4. Run in Command Line and accept fix all the issues (accept all conditions)
   ```sh
   flutter doctor
   ```
5. Install [Android Studio](https://developer.android.com/studio)
