## Getting Started

This is how you set up this project locally.
To get a local copy up and running follow these simple steps.

### Prerequisites

* Git (Windows)
  ```sh
  got to https://git-scm.com/downloads and install it
  ```
* Git (Linux, Unix -Debian/Ubuntu-)
  ```sh
  apt-get install git
  ```
  * Git (Linux, Unix -Ubuntu-)
  ```sh
  add-apt-repository ppa:git-core/ppa
  ```
  * Git (Linux, Unix -Fedora-)
  ```sh
  add-apt-repository ppa:git-core/ppa
  ```
  
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/flutter/flutter.git -b stable
   ```
3. Edit [environment variables](https://stackoverflow.com/questions/60212125/flutter-install-in-windows-environment-variable) for your account 
4. Run in Command Line and accept fix all the issues (accept all conditions)
   ```sh
   flutter doctor
   ```
5. Install [Android Studio](https://developer.android.com/studio)
